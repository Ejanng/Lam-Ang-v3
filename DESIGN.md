# Game Design Document
# Lam-Ang Chronicles

## Overview
An educational RPG that teaches players about "Biag ni Lam-Ang," an Ilocano epic poem, through interactive storytelling and gameplay.

## Target Audience
- Students (ages 10+)
- Educators teaching Philippine literature
- Anyone interested in Filipino culture and mythology

## Core Gameplay Loop
1. Player selects a chapter
2. Reads story content with educational points
3. Completes a quiz to test comprehension
4. Earns experience and unlocks next chapter
5. Progresses through all 5 chapters

## Learning Objectives
- Understand the plot and characters of Biag ni Lam-Ang
- Learn about pre-colonial Filipino culture
- Appreciate Ilocano traditions and values
- Recognize themes in epic literature
- Connect with Philippine cultural heritage

## Game Systems

### Story System
- 5 main chapters following the epic's narrative
- Rich text descriptions
- Educational commentary
- Cultural context

### Quiz System
- Multiple choice questions
- Immediate feedback
- Explanations for correct answers
- Points awarded for correct answers

### Progression System
- Experience points (XP)
- Level ups
- Chapter unlocking
- Achievement tracking

### Stats System
- Health
- Strength
- Intelligence
- Wisdom
- Charisma

### Save System
- Auto-save on progress
- Manual save option
- Load game on continue

## UI/UX Design

### Main Menu
- Clean, welcoming interface
- Clear navigation
- Educational theme

### Chapter Select
- Visual progress indicators
- Locked/unlocked chapters
- Chapter descriptions

### Story Display
- Readable text layout
- Educational points highlighted
- Clear progression options

### Quiz Interface
- Clear question display
- Easy option selection
- Immediate feedback
- Encouraging messaging
