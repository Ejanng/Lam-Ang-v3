# Quick Start Guide - Lam-Ang Chronicles

## Getting Started

### Prerequisites
- **Godot Engine 4.2 or higher** - Download from https://godotengine.org/

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Ejanng/Lam-Ang-v3.git
   cd Lam-Ang-v3
   ```

2. **Open in Godot:**
   - Launch Godot Engine
   - Click "Import"
   - Navigate to the cloned folder
   - Select `project.godot`
   - Click "Import & Edit"

3. **Run the game:**
   - Press **F5** or click the **Play** button in Godot

## How to Play

### Main Menu
- **New Game**: Start a fresh playthrough
- **Continue**: Resume from your last save
- **About**: Learn about the Biag ni Lam-Ang epic
- **Quit**: Exit the game

### Playing Through Chapters

1. **Select a Chapter** from the chapter selection screen
   - Initially, only Chapter 1 is unlocked
   - Complete chapters to unlock the next ones

2. **Read the Story**
   - Read through the narrative of Lam-Ang's adventures
   - Pay attention to the educational points
   - These highlight important cultural and historical information

3. **Take the Quiz**
   - Click "Take Quiz" after reading
   - Answer the question about the chapter
   - Get immediate feedback and explanation

4. **Complete the Chapter**
   - Click "Complete Chapter" to finish
   - Earn experience points
   - Unlock the next chapter

### Game Progression

- **Experience Points**: Earned by completing chapters and quizzes
- **Levels**: Automatically gain levels as you earn experience
- **Stats**: Your strength and intelligence increase with each level
- **Completion**: Track your progress through all 5 chapters

## The Story

### Chapter Overview

1. **The Birth of Lam-Ang**
   - Learn about Lam-Ang's miraculous birth and special powers
   - Discover why he's different from ordinary heroes

2. **The Quest for Don Juan**
   - Follow Lam-Ang as he searches for his father
   - Meet his loyal companions: the magical rooster and dog

3. **The Courtship of Ines Kannoyan**
   - Experience Lam-Ang's quest for love
   - Learn about traditional Filipino courtship

4. **The Great Trial**
   - Witness Lam-Ang's incredible feats
   - Understand the supernatural elements of the epic

5. **Death and Resurrection**
   - Experience the climactic events
   - Learn about themes of loyalty and rebirth

## Tips for Success

### For Students
- **Read carefully**: The educational points contain important information
- **Take notes**: Consider writing down key facts about Filipino culture
- **Don't rush**: Take time to understand each chapter

### For Teachers
- **Classroom Use**: Great for teaching Philippine literature

## Controls

### Menu Navigation
- **Mouse**: Click buttons to navigate
- **Keyboard**: Use Tab to move between buttons, Enter to select

### Exploration Mode (if playing game world)
- **WASD** or **Arrow Keys**: Move character
- **E** or **Space**: Interact with objects/NPCs
- **ESC**: Open menu
- **J**: Attack (in combat)

## Save & Load

- **Auto-Save**: Progress is automatically saved after completing chapters
- **Manual Save**: Use the menu to save anytime
- **Continue**: Resume from your last save point
- **Save Location**: Saves are stored in your user directory

## Educational Value

This game teaches:
- **Philippine Literature**: One of the most important Ilocano epics
- **Cultural Heritage**: Pre-colonial Filipino traditions and values
- **Historical Context**: Life in ancient Ilocos region
- **Literary Elements**: Epic poetry, hero's journey, supernatural elements
- **Values**: Family, bravery, perseverance, honor

## Troubleshooting

### Game won't start
- Ensure you have Godot 4.2+ installed
- Check that all files were properly cloned
- Try reimporting the project in Godot

### Quiz not working
- Make sure you've selected an answer before clicking Submit
- Check that the game_data.json file is in the data folder

### Save file not loading
- Check that you have write permissions in the user directory
- Try starting a new game if the save is corrupted

### Missing content
- Verify all files were downloaded correctly
- Reimport the project in Godot
